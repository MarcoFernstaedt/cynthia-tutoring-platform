# Saguaro Blossoms Learning Services

A polished one-page website concept for Cynthia’s tutoring and learning-services business.

## What is included

- Next.js + Tailwind landing page
- Services, audiences, philosophy, About/Why copy, pricing, and contact-intake section
- Content contract test to protect required Cynthia source copy and UI signals
- Desktop/mobile walkthrough media in `walkthrough/`
- GitHub Pages deployment workflow

## Local commands

```bash
npm test
npm run lint
npm run build
npm run dev
```
